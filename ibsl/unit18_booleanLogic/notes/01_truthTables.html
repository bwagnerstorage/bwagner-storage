Boolean Operators

Boolean Logic, is a subset of algebra used for creating true/false statements. Boolean expressions use the operators AND, OR, XOR, NOT, NAND, and NOR to compare values and return a true or false result.

These boolean operators are described in the table below:

Operator IB Description
AND x AND y returns TRUE if both x and y are true;
returns FALSE if either x or y are false.
OR x OR y returns TRUE if either x or y, or both x and y are true;
returns FALSE only if x and y are both false.
XOR x XOR y returns TRUE if only x or y is true;
returns FALSE if x and y are both true or both false.
NOT NOT x returns TRUE if x is false;
returns FALSE if x is true.
NAND (NOT AND) x NAND y returns TRUE if either x or y, or both x and y are false;
returns FALSE only if x and y are both true.
NOR (NOT OR) x NOR y returns TRUE if both x and y are false;
returns FALSE if either x or y, or both x and y are true

Truth Table
