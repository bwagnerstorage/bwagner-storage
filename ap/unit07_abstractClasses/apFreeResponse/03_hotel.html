Directions

A set of classes is used to represent the type of rooms that are available at a certain hotel. The class hierarchy is shown in the following diagram.

All rooms have a room number, room rate, and type of bed (double, queen, king). The class Suite is specified as an abstract class as shown in the following declaration

public abstract class Suite
{
    private int roomNumber;      // room number

    
public Suite(int room)

    {
        roomNumber = room;
    }

    // returns the room rate for this suite
    
public abstract double getRoomRate();


    // returns the bed type for this suite
    
public abstract String getBedType();


    // returns a string with information about the suite
    
public String toString()

    {
        return "Room number: " + roomNumber + "\n" +
               "Room rate  : " + getRoomRate() + "\n" +
               "Bed type   : " + getBedType();
    }
}

Each room is assigned a room number through the parameter list of the constructor. For all suite classes, the toString method returns a string containing information for that suite. Three additional classes are used to represent the different types of suites and are described in the table below.

Using the class hierarchy and specifications given above, you will write complete class declarations for the DeluxeSuite and HoneymoonSuite classes.

  1. Write the complete class declaration for the class DeluxeSuite. Include all necessary instance variables and implementations of its constructor and method(s). The constructor should take a parameter that indicates the room number for the suite being reserved. The cost of a suite is $100.00 per night and it includes a king size bed.

  2. Write the complete class declaration for the class HoneymoonSuite. Include all necessary instance variables and implementations of its constructor and method(s). The constructor should take a parameter that indicates the room number for the suite being reserved. The toString method should include a notation that the HoneymoonSuite includes two white terrycloth robes and free room service. A HoneymoonSuite room rate is the DeluxeSuite room rate plus $25.00. If the room rate for the DeluxeSuite changes, the HoneymoonSuite room rate should continue to be computed correctly with no code modifications to the HoneymoonSuite. The HoneymoonSuite bed type is the same as the DeluxeSuite.
