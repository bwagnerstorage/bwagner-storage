  1. Write the GroceryStore method setPrice. This method updates the price of the given item in the groceryList to the specified price. If the itemName is not in the list no changes are made.

    For reference, assume groceryList contains the following grocery items.

    Name Price Size (in ounces) Category
    avocado 1.68 8 P
    milk 1.92 64 D
    chicken 4.48 64 M
    broccoli 1.92 16 P
    yogurt 0.96 16 D
    spinach 1.76 16 P
    cornedbeef 6.72 48 M
    porkchops 2.24 32 M

  2. The unit price of an item is the price per ounce.

    Name Price Size (in ounces) Category
    avocado 1.68 8 P
    milk 1.92 64 D
    chicken 4.48 64 M
    broccoli 1.92 16 P
    yogurt 0.96 16 D
    spinach 1.76 16 P
    cornedbeef 6.72 48 M
    porkchops 2.24 32 M

    The unit price of avocado is 1.68 divided by 8, which equals 0.21, and the unit price of spinach is 1.76 divided by 16, which equals 0.11.

    Write the GroceryStore method bargainItem. This method returns the name of an item whose unit price is the lowest in the specified category. If there is more than one item with the lowest price, any one of these items may be returned. If there are no items in the category, bargainItem returns "none".

    For example, consider the items and prices listed in the table above. Using this table, the results of three calls to bargainItem are shown below.

                  Method call                 Returned value
            bargainItem("P")             spinach

            bargainItem("M")             chicken or porkchops

            bargainItem("B")             none

    In writing bargainItem, you may call any of the member methods of the GroceryStore class. Assume that the member methods work as specified.
