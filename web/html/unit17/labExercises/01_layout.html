Lab Exercise

  1. Open Notepad and create a new document named 17_1_layout.html and save it to your HTML folder.

  2. Give the document the title: 17_1_layout.

  3. Use the HTML 5 layout elements header, footer, nav, section, and aside to create the layout shown below.

  4. Use internal styling to format the elements.

  5. Use any color you like for the different elements.

  6. Use the h3 tag to label each section.

  7. The header and footer elements should have a height of 100px. The nav element should have a height of 50px.

  8. The section element should have a width of 75%. It should also include the following attribute: float:left.

  9. The aside element should include the following attribute: overflow:hidden.

  10. The section and aside elements should have a height of 300px.

  11. The h3 tag will probably force the different sections to be spaced apart. To fix this style the h3 element with the following attribute: display: inline-block.
