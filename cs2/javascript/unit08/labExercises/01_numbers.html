    Number data retrieved from a text box are always of type String. In order to use the data in a mathematical expression the data must be converted to a Number. This can be accomplished with the Math class's parseInt method. For example, suppose your code contains a variable named data that contains the String value "99". The following code will convert the data from the String "99" to the Number 99:

    var num = parseInt(data);
