Lab Exercise

  1. File Name: 6_1_events.html

  2. Copy the following code:

    <!DOCTYPE html>
<html>
    <head>
        <title>6_1_events</title>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
    </head>
    <body>
        
        <script>
            function changeBackgroundColor()
            {
                var r = Math.floor(Math.random() * 256);
                var g = Math.floor(Math.random() * 256);
                var b = Math.floor(Math.random() * 256);
                
                var color = "rgb("+ r +","+ g +","+ b +")";
                
                document.body.style.backgroundColor = color;
            }
        </script>
    </body>
</html>

  3. The function changeBackgroundColor defined above creates a random color then sets that color as the body element's background color.

    Add a button labeled "Change Color" to the page that calls changeBackgroundColor when it is clicked.
