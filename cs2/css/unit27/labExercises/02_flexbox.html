Lab Exercise

  1. Open VS Code and create a new document named 27_2_flexbox.html and save it to your CSS folder.

  2. Give the document the title: 27_2_flexbox.

  3. In VS Code create a file named 27_2_flexbox.css and save it to your CSS folder.

  4. Use external styling to create the calculator interface shown below.

  5. Your design should include a main flex container that contains six flex containers.

  6. Use the following style properties/attributes:

    • Calculator - width: 250px, height: 400px, padding: 20px;

    • Calculator output screen - min-width: 250px, height: 40px

    • Calculator buttons - width: 50px, height: 50px

    • Last two buttons - width: 100px

    • Use box-sizing: border-box for all elements.

      * { box-sizing: border-box; }
