Lab Exercise

We are going to change things up a little bit in this exercise. We have been using W3schools.com as our primary source, but we are now going to change to a different website Mozilla.com. These are the people who make the FireFox browser. W3schools is a great site, but I think Mozilla does a better job of explaining the techniques available for laying out a website. Webpage Layout refers to the organizing and placement of the different sections or elements of a webpage.

  1. Click on the following link: CSS Flexbox

  2. Read the tutorial entitled Flexbox to learn how this CSS attribute can be used to organize elements on a page horizontally or vertically.

  3. When you have a good understanding of how Flexbox works, then click on this link CSS Flexbox Test Your Skills to practice the skills you have learned by completing the exercises Flex Layout One, Flex Layout Two, Flex Layout Three, and Flex Layout Four.

  4. You have the option of completing the exercises online in the provided coding window



    or you can click the link in the yellow box that opens a file in the GitHub environment. The code in this file can then be copied to a file in your VSCode environment.
