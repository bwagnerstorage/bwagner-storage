    This html document contains 6 div boxes that have similar styling because they all use the styling provided by the allDivs class. The allDivs class specifies that each div's top, right, bottom, and left margins all have a distance of 20px. In this lesson, you are going to use the id property to customize the margins of each of the 6 boxes.

  4. d1: Change the left margin's value to 100px.

  5. d2: Change the top margin's value to 50px and the bottom margin's value to 100.

  6. d3: Use the margin property to change all four margin's (top, right, bottom left) to a value of zero. Notice that although the left margin of the d3 box is zero there is still a gap between the left side of the div and left edge of the browser window. The reason for this gap is that the body element has default margin sizes. In the body tags styling section set all of its margins to zero. This should remove the gap between the div and left edge of window.

  7. d4: Use the margin property to set the top, right, and bottom margins to 10px and set the left margin to 400px.

  8. d5: Use the margin property to set both the top and bottom margins to 5px.

  9. d6: Use the margin property to set the margins to auto. The auto property will center an element horizontally within its container. The element will then take up the specified width, and the remaining space will be split equally between the left and right margins. This means that in order for the auto property to work the element must be given a width using the width property.
