CSS Selectors

CSS selectors allow you to choose the content you would like to style in a html document. CSS selectors select HTML elements according to their id, class, type, etc.

There are several types of selectors in CSS. In this lesson, you will look at the five "simple" selectors:

  • CSS Universal Selector

  • CSS Element Selector

  • CSS Group Selector

  • CSS Id Selector

  • CSS Class Selector

To illustrate the different types of selectors you are going to create the following table using each of the selectors listed above.

    Universal Selector

    The Universal selector (*) allows you to select and apply styling to every element on a page.

    Copy the CSS code below into the html document's style section under the heading /* Universal Selector */.

    * {
   font-family: Georgia, 'Times New Roman', Times, serif;
}

    This code sets the font of every element in the document to Georgia. If the Georgia font is not available then it set the font to Times New Roman, and so forth.

    Element Selector

    The Element selector allows you to select specific elements within the document to apply styling.

    Copy the CSS code below into the html document's style section under the heading /* Element Selector */.

    table {
   width: 70%;
   margin-left: auto;
   margin-right: auto;
}
caption {
   font-size: 1.6rem;
   padding: 10px;
}
td { 
   padding: 8px;
   color:aliceblue;
}

    This code provides styling for any table, caption, or td element placed within the document.

    Group Selector

    The Group selector allows you to apply the same styling to several elements without having to repeat the code.

    Copy the CSS code below into the html document's style section under the heading /* Group Selector */.

    table, tr, td, th {
   border-collapse: collapse;
   border: 2px solid steelblue;
   background-color:black;
}

    This code provides the same styling for the table, tr, td, and th elements.

    Selector Ordering

    The order the CSS selectors are listed in the style section is important. The browser reads the selectors in sequential order from top to bottom.

    Copy the CSS code below into the html document's style section under the heading /* Order of selectors is important */.

    th {
   font-size: 1.2rem;
   color:black;
   background-color: lightsteelblue;
}

    This code provides styling for the th element. Since the Group selector shown above set the background color of th tag to black, I had include another Element selector for the th tag below the Group Selector so that I could change the background color.

    Class Selector

    The Class selector allows you to apply styling to all the HTML elements that have been assigned to the same class. An element is assigned to a class using its class attribute.

    Copy the CSS code below into the html document's style section under the heading /* Class Selectors */.

    .myClass {
   font-family: Consolas, sans-serif;
   font-size: 0.85rem;
   color:lightseagreen;
}

    This code provides styling for all the td elements assigned to the myClass class.

    ID Selector

    The ID selector allows you to apply styling to one element with an HTML document.

    Copy the CSS code below into the html document's style section under the heading /* ID Selectors */.

    #myId {
   color: red;
}

    This code provides styling for the td element assigned the id myId.

    An id needs to be unique and only assigned to one HTML element within a document.

    Note: You will use the id attribute a lot when you start programming in Javascript.
