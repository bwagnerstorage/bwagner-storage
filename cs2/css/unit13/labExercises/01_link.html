  4. Add the text-decoration property to the anchor tag's styling. This property, when set to "none", removes the underline that usually appears when an anchor tag is used.

    text-decoration: none;

    5. Link State

    Links can be styled differently depending on what state they are in. A link can be in one of the four states listed below:

    • a:link - a normal, unvisited link
    • a:visited - a link the user has visited
    • a:hover - a link when the user mouses over it
    • a:active - a link the moment it is clicked

    When setting the style for several link states, there are some order rules:

    • a:hover MUST come after a:link and a:visited
    • a:active MUST come after a:hover
