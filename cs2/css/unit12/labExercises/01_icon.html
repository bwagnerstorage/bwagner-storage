    • Introduction

    In this exercise you are going to learn how to add icons to your web pages. As with fonts, there are two methods used for adding icons:

    • You can download the icons from an icon repository or library like fontawesome.com. Once downloaded the icons can then be added to your html documents using the <i> tag and styled using css.

    • You can add a link to your web page that references the icon's files directly from an icon library or repository like FontAwesome. This technique does not require any downloading. As with the first option, you can use the <i> tag and css to style the icons.
