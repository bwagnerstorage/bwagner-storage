Chapters

  • CSS Tables

Lab Exercise

  1. Open VS Code and create a new document named 15_1_table.html and save it to your CSS folder.

  2. Give the document the title: 15_1_table.

  3. In VS Code create a file named 15_1_table.css and save it to your CSS folder.

  4. Add the following starter code to your html document:

    <!DOCTYPE html>
   <html>
   <head>
       <title>Table</title>
       <style>
          
       </style>
   </head>
   <body>
      <table>
         <caption>US Presidents 1977 - Present</caption>
         <tr>
            <th>Name</th><th>Term</th><th>Party</th><th>Home State</th>
         </tr>
         <tr>
            <td>Jimmy Carter</td><td>1977-1981</td><td>Democratic</td><td>Georgia</td>
         </tr>
         <tr>
            <td>Ronald Reagan</td><td>1981-1989</td><td>Republican</td><td>California</td>
         </tr>
         <tr>
            <td>George H.W. Bush</td><td>1989-1993</td><td>Republican</td><td>Texas</td>
         </tr>
         <tr>
            <td>William Clinton</td><td>1993-2001</td><td>Democratic</td><td>Arkansas</td>
         </tr>
         <tr>
            <td>George W. Bush</td><td>2001-2009</td><td>Republican</td><td>Texas</td>
         </tr>
         <tr>
            <td>Barack Obama</td><td>2009-2017</td><td>Democratic</td><td>Illinois</td>
         </tr>
         <tr>
            <td>Donald Trump</td><td>2017-present</td><td>Republican</td><td>New York</td>
         </tr>
      </table>
   </body>
   </html>

  5. Apply the following styling to the table:

    • Set the font-size property of the table's caption to 1.2em.

    • Set the border property of the table, th, td elements to a size of 1px, border-style of solid, and color black.

    • Set the border-collapse property of the table to collapse.

    • Set the font-size property of the table to 1.2em.

    • Set the height property of the th element to 40px.

    • Set the font-size property of the th element to 1.4em.

    • Set the text-align property of the th element to center.

    • Set the vertical-align property of the th element to middle.

    • Set the color property of the th element to white.

    • Set the background-color property of the th element to rgb(12,12,68).

    • Set the padding property of the th and td elements to 15px.

    • Use the nth-child selector to create a zebra style table that changes the background color of all of the even rows in the table to the following color: #dddddd.

    • Use the hover selector on the tr element so that the background color of a row in the table will change colors when the mouse hovers over it. Use the following color: rgb(129,175,206).

  6. Center the table horizontally within the document by adding the following code to the table element's styling. 

    margin-left: auto; 
margin-right: auto;
