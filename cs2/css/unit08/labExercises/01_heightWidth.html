    Height and Width Properties

    The height and width properties are used to set the height and width of an element.

    The height and width properties do not include padding, borders, or margins. It sets the height/width of the area inside the padding, border, and margin of the element.

  10. Set the background color of the orange div to orange. Set the width to 100vw and the height property to 100vh. View the page in a browser.

    The word viewport refers to the browser window size. The unit of measurement, vw, stands for viewport width and is equal to 1% of the width of the viewport or browser window. Therefore, 10vw is equal to 10% of the width of the viewport and 100vw is equal to 100% of the width of the viewport. The unit of measurement, vh, is equal to 1% of the height of the viewport or browser window. Therefore, 10vh is equal to 10% of the height of the viewport and 100vh is equal to 100% of the height of the viewport.
