    6. Multiple Background Images

  6. background.image - Add the following CSS code to the body section of the internal styling:

    background-image:url(images/texas_flag.png), url(images/sandstone.jpg);

    This code assigns two images as the background of the documents body element. When multiple images are assignment to a background the images are stacked on top of each other. The image listed first is the image that will be on the top of the stack of images and therefore will be displayed on top of the other images.

  7. background-position - Add the following css code to the body section of the internal styling:

    background-position: top right, top left;

    This code positions the images within the body. The first pair of words, top and right instruct the browser to position the image at the top right corner of the body.

    The second pair of words, top and left instruct the browser to position the image at the top left corner of the body.

  8. background-repeat - Add the following css code to the body section of the internal styling:

    background-repeat:no-repeat, repeat;

    This code determines whether or not the image will be repeated to cover the entire window. The flag will not be repeated but the sandstone image will.

  9. background-attachment - Add the following css code to the body section of the internal styling:

    background-attachment:fixed, scroll;

    This code determines whether an image will be in a fixed position on the screen or will move when the document is scrolled. The code is currently set to fix the flag image at its location so that it will not move during scrolling and the sandstone image is set to scroll so it will move as the window is scrolled.

    View the document in your browser. Notice how the text scrolls over the top of the background images. Since the sandstone image is set to repeat. When the page is scrolled new sandstone images are added to the window so that the background is always covered with the image.

Modifications

In this section, the texas_flag image is going to be moved to the center of the browser window. It is also going to be stretched to cover the window's entire background. Finally, the image's opacity is going to be modified by applying one of the blend-mode options to the two backgrounds.

  1. Change the flag's attachment property back to fixed.

  2. Change the background-position property to the following:

    background-position: center center, top left;

    This code changes the flag image's position so that it is located in the center of the window.

  3. background-size - Add the following css code to the body section of the internal styling:

    background-size: cover;

    This code stretches the two images so that they completely cover the body's background.

  4. background-size - Add the following css code to the body section of the internal styling:

    background-blend-mode: soft-light;

    This code determines how an element's background images should blend with each other and with the element's background color.
