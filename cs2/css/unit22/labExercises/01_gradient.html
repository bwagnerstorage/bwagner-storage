Lab Exercise

  1. Open VS Code and create a new document named 22_1_gradient.html and save it to your CSS folder.

  2. Give the document the title: 22_1_gradient.

  3. Use internal styling to create a document like the one shown below.

  4. The body element should have a gradient of your choosing that covers the entire window. In the center of the window should be a div with a border with a gradient of your choosing that covers its background.

  5. To center the div within the document window use the following styling:

    margin: auto;
position: absolute;
top: 0; left: 0; bottom: 0; right: 0;

  6. Don't forget to give the div a width and height.
