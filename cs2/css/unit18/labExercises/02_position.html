Position

In this exercise, the position property's fixed value is used to position a vertical menu of a web page by providing its location within the browser window.

    5. Fixed Positioning

    To demonstrate fixed positioning you are going to add a vertical menu to the body section of the page. Fixed positioning allows you to specify exactly where an element is positioned relative to the browser window.

  5. Add the following internal styling to the style section of your page. Notice the use of the position property with a value of fixed.

    #menu {
   position: fixed;
   margin: 0;
   left: 5px;
   font-size: 1.8em;
   height: 100%;
   background-color: #CCCC;
}

    The css above will position the container menu on the left side of your page stretching from the top of the page down to the bottom. It's position is fixed at the top of browser window 5 pixels from its left edge.

  6. View the web page in the browser. Scroll the window on the right. Notice how the menu on the left does not move. It is fixed into place. Fixed positioning allows you to place an element directly on the browser window by providing its location relative to the window. For example, top:0 and left:0 would be located at the top right corner of the window. By choosing top:8px and right:15px the element is positioned 8 pixels from the top of the window and 15 pixels from the window's right edge.(Typically when deciding on a location, you would choose either top or bottom and then left or right.)
