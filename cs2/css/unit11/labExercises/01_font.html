    4. In this exercise, you are going to explore the properties/attributes of fonts. In order to use a font on a webpage the font must be installed on the computer that is displaying the page or there must be a link on the web page to a location on the Internet where information about the font resides.

    A web safe font is a font that is considered to be a 'safe bet' to be installed on the vast majority of computers. Every computer that has a browser installed has default fonts built in so that it can display the text on web pages.

    A font family is a set of fonts that have a common design. To create a font family in css make a list of fonts, separated by commas. Start with the font that you really want to use and then add to the list other fonts that are similar but not your first choice. End the list with a generic font that the browser can use as a last resort (i.g. serif, sans-serif).

    Serif fonts are usually easier to read in extended writing portions of a web page than sans-serif fonts. This is because serif fonts make the individual letters more distinctive and easier for our brains to recognise quickly.

  8. Add the following styling to the paragraph with the id p5:

    Change the size of the font using the font-size property and the value 1.8em.

    The em size unit allows users to resize the text of web pages using the browser(i.e zoom). For this reason many developers are now using em instead of pixels. 1em is equal to the current font size. The default text size in browsers is 16px. So, the default size of 1em is 16px.

  9. Add the following styling to the paragraph with the id p6:

    Change the size of the font using the font-size property and the value 2vw.

    The vwunit is a responsive size unit. This means that the font size will change as the browser's viewport increases or decreases in size. Viewport is the browser window size. 1vw = 1% of viewport width.
