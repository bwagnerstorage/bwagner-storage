    4. CSS Text Properties

    In this exercise you learn about the css properties that allow you to modify text. Some of these you have already encountered in other lessons. Follow the directions below to see some of these properties in action.

  4. Add the following styling to the h1 tag:

    • Use the font-size property with a value of 3.0em.
    • Use the text-align property to center the text horizontally within the h1 element.
    • Use the text-shadow property with the following attributes: 3px 3px 3px gray.

  5. Add the following styling to the paragraph with the id p1:

    • Use the color property to set the text color to blue.
    • Use the text-align property with the justify attribute so that the text is aligned on both the left and right margins of the paragraph.

    7. Use the text-decoration property with the underline attribute so that the paragraph's text is underlined.
