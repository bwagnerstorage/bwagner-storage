Lesson

In this lesson you are going to learn how to enhance the capabilities of VS Code by installing extensions.

The features that Visual Studio Code includes out-of-the-box are just the start. VS Code extensions let you add UI themes, languages, and tools to your installation to support your development. If you would like more information about extensions click on the following link: VSCode Extensions

Open Extensions View

You can browse and install extensions from within VS Code. Bring up the Extensions view by clicking on the Extensions icon in the Activity Bar.

By default, Extensions displays all of the extensions currently installed. There are other options available under the More Options Menu (three dots). For example, Show Built-in Extension displays the extensions that come pre-installed.

.

Installing Extensions

As an example of how to install an extension lets install a new Color theme.

  1. In the search box, replace @install with the word "theme". This will automatically update the view with all the extensions that contain the word "theme".

  2. Browse the list and find a theme you would like to install and click the green install button.


Change Color Theme

Click the Manage button located at the bottom of Activity Bar, and select the Color Theme option from the menu.

Use the Up and Down arrow keys to browse the installed Color Themes. If you press the Esc key the window will close and the theme will not be changed.
