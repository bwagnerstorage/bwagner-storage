Lab Exercise

  1. Open VS Code and create a new document named 20_1_video.html and save it to your HTML folder.

  2. Give the document the title: 20_1_video.

  3. Use HTML5 video elements to play the following video within your document.

  4. Create a new folder inside your HTML folder named videos.

  5. Right-click the following link and save the mp4 file to your videos folder: Bugs and Daffy.mp4

  6. Video Attributes: filename = Bugs and Daffy.mp4, width = 480px, height = 360px

  7. Use the control attribute so the controls will be displayed when the video is playing.
