Lesson

An editor is a computer program that enables a user to add or edit text. Editor's can be as simple as NotePad or as complex as Microsoft Word. In the world of web design, there are editors specifically designed to make the creation of web documents easier and faster. Here is a list of some of the most popular of today's editors.

  • NotePad++

  • Sublime Text

  • Atom

  • Vim

  • Visual Studio Code

  • Brackets

All of the editors listed are free to use except for Sublime Text. Currently my favorite editor is Visual Studio Code developed by Microsoft, but that's just my preference.


Online editors have also become popular in recent years. These editors run in your browser so there is no need to install anything on your computer. These online editors are nice because they can be accessed from anywhere, there drawback is they lack many of the advanced features that you get with desktop editors.

Because of our need current needs for virtual learning we will be using one of these online editors named Code Pen for this course. This will prevent us from having to install and setup software on your computer, however, if you would prefer to use one of the desktop editors like Visual Studio Code, be my guest. Another popular online editor is named JSFiddle. You might want to check it out as well.

Code Pen setup

  1. Type this URL in your browser, http:\\codepen.io or just click the following link: Code Pen.

  2. Click on the green button "Sign Up for Free".

  3. When the next screen appears click on the button "Sign Up with Email".

  4. The screen will expand below the button so as to provide you with fields where you can type iin your name, username, and email address (Use your school email). Finally select a password that you will remember and click on the green Submit button.

  5. Congratulation! You now have a Code Pen account, so login and look around.

  6. Question: What is a pen? In Code Pen, a pen consists of three windows. One window for typing HTML code, a second window for adding some CSS code, and finally a third window for adding some Javascript code. You will learn about each of these as you work your way through the course.

  7. Question: What is the difference between a project and a pen. In Code Pen, projects are more advanced and allow for multiple files of each of the three types. Since we are using the free version, we will always use penswhich only allow one file per window or pen.
