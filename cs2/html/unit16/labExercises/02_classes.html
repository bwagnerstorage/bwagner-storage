Lesson

In this lesson you are given a list of the last 10 US presidents and their party affiliation. Depending upon a president's party affiliation you are going assign either the democrat class or the republican class as an attribute of a div element that you will be creating for each president.

Lab Exercise

  1. Open VS Code and create a new document named 16_2_classes.html and save it to your HTML folder.

  2. Give the document the title: 16_2_classes.

  3. Add essential HTML starter code to the file.

  4. Use an h1 element to display the following heading at the top of the page's body:
    US Presidents (1963-present)

  5. Below the heading, create a div element for each of the ten US presidents listed below.

    US Presidents (1963-present)
    NameParty Affiliation
    Lyndon JohnsonDemocrat
    Richard NixonRepublican
    Gerald FordRepublican
    Jimmy CarterDemocrat
    Ronald ReaganRepublican
    George H. BushRepublican
    Bill ClintonDemocrat
    George W. BushRepublican
    Barack ObamaDemocrat
    Donald TrumpRepublican

    Add each of their names to a div like the example shown below.

          <div>Lyndon Johnson</div>


  6. Add
    a class attribute for each of the presidents. Assign them a class name according to their party affiliation. If they are listed in the table as a Democrat then assign them the class name democrat and if they are listed as a Republican then assign them the class name republican.

  7. Display the web page in the browser. The border's of each president's div should be colored either blue or red depending upon their party affiliation.
