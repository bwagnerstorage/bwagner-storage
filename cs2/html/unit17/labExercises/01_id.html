17-1 HTML IDs

Resources

  • HTML IDs

Lesson

In this lesson, you will learn how to assign an identifier to a HTML element using the id attribute. In contrast to the class attribute used in the previous unit, the identifier assigned to the id attribute is unique and can only be assigned to one HTML element. Both CSS and JavaScript can use this identifier to perform certain tasks that are unique to that element.

The id attribute uses the same format as other HTML attributes.

<div 
id
="myIdName">
   
   
</div>

  3. Display the html file in the browser. Notice the three blocks all have individualized styling including different sizes, border color, font color and font size. This unique styling is accomplished by assigning each div its own id using the id attribute like the one shown below.

    <div id="blockA">A</div>

  4. Using the id attribute the div is assigned the id name "blockA". This identifier uniquely identifies this div in the code by this name. This id can only be assigned to one html element. Now look at the CSS code that is assigned to this id using this id. 

    #blockA{
   width: 100px;
   height: 100px;
   background-color:beige;
   color:darkred;
   border: 4px solid darkred;
   font-size:5.5rem;
   text-align: center;
   box-shadow: 4px 4px 10px rgb(126, 109, 109);
   margin-right: 15px;
}

  5. In the CSS, the hash tag symbol "#" is used to assign styling for the html element with the specified id.

  6. To see how this works, add a new div element to the html document, below the "blockC" div, that assigns the id blockD to the new div. Type the letter "D" between the div's opening and closing tags.

    7. View the page in the browser. A new block labeled "D" should now appear with its own unique styling. This styling was specified in the CSS styling section using #blockD.
