Random Ranges

Directions

Write a program that uses the Math class's random method to create random numbers using the following criteria:

  1. Create a random integer in the range of 0 to 25 inclusive.
  2. Create a random integer in the range of 1 to 3 inclusive.
  3. Create a random integer in the range of 50 to 100 inclusive.
  4. Create a random integer in the range of -10 to -1 inclusive.
  5. Create a random integer in the range of -100 to 100 inclusive.
  6. Create a random character in the range of A to Z. To do this first generate a random number in the range of 65(A) to 90(Z): 
    int num6 = (int)(Math.random() * 26) + 65;
    then when you print cast the number to a character using a character cast (char).
    System.out.println((char)num6);

    Programming Challenge

  7. Create a random real number in the range of 0 to 1 exclusive displayed with a maximum of three decimal places (.235, .523, .661, .8, .45, etc).
    (Hint: create a random integer in range of 0 to 999 then move the decimal point 3 places to the left.
  8. Create a random integer in the range of 1000 to 10000 inclusive displayed to the nearest thousand (1000, 2000, 3000, etc).
