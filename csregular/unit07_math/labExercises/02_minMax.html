MinMax

Directions

Write a Java program that allows a user to enter in three integer values from the keyboard. Use the Math class's min method to determine which of the three values is the smallest. Use the Math class's max method to determine which of the three values is the largest. Print the results in the format shown in the Sample Run.

Note: the min and max methods can only compare two values at one time.

Source File

MinMax.java

Sample Run

Enter 1st Integer --> 20
Enter 2nd Integer --> 7
Enter 3rd Integer --> 95

Smallest = 7
Largest =  95
