Rectangle

Directions

Write a Java program that prompts the user for the dimensions of a rectangle, then calculate and print the perimeter and area of the rectangle. Use the following rectangle formulas:

   Perimeter = 2L + 2W
   Area = L * W

You will need to use the Scanner class's nextInt() method in your implementation.

Source File

Rectangle.java

Sample Run

Enter the Length --> 10
Enter the Width --> 20

Perimeter = 60
Area = 200
