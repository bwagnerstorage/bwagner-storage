ExamScores

Directions

Write a program that allows a user to enter exam scores in the range of 0 to 100. The user should be allowed to enter scores until he/she types the sentinel value -1. The program should then tally the total number of scores entered and the number of As, Bs, Cs, Ds, and Fs according to the following scale.

Range Grade
90 to 100 A
80 to 89 B
70 to 79 C
60 to 69 D
0 to 59 F

The sentinel value is used only to end the loop, so do not use it in the calculations.
