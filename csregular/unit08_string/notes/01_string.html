String Class

Strings are sequences of characters, such as "Hello" or "have a nice day". Java does not have a built-in string type. (Recall the eight primitive types: boolean, byte, char, double, float, long, int, short.) Instead, the standard Java library contains a predefined class called, naturally enough, String. String objects are unique in that there are two ways they can be created or instantiated. The first method uses the new operator and is the same technique used to instantiate other Java objects.

Method 1 - Instantiating String Object

String str = new String();   
// instantiate String object

str = "Good Morning";        
// assign string value to String object

System.out.println(str);

Method 2 - Instantiating String Object

The second method is simpler and more convenient since it uses the same technique used to declare primitive variables.

String str;              
// declare String object

str = "Good Morning";    
// assign string value to String object

System.out.println(str);

Method 3 - Instantiating String Object

The third method is similar to the technique used to initialize primitive variables.

String str = "Good Morning";   
// initialize String object with string value

System.out.println(str);

Since the String class is used in most Java programs, the designers of Java decided to allow programmers the ability to instantiate String objects without having to use the new operator. However, it must be understood, that String is a class, not a primitive data type, therefore String objects can be sent messages. Below are a few examples of String methods that can be used to communicate with String objects:

str.length()
str.substring(0,4)
str.equals("married")
str.indexOf(".")

Concatenation

Java, like most programming languages, allows you to use the + sign to join (concatenate) two strings together.

String firstName = "John";
String lastName = "Wayne";
String fullName = firstName + " " + lastName;

The above code makes the value of the String variable fullName "John Wayne". Notice how I inserted a space between the names. Alternatively, I could have done this by including the space with the names such as "John " or " Wayne".

When you concatenate a string with a value that is not a string, the latter is converted to a string. (As you will see later, every Java object can be converted to a string.) For example:

int age = 13;
String rating = "PG" + age;

sets rating to the string "PG13". This feature is commonly used in output statements, for example,

int answer = 128;
System.out.println("The answer is " + answer);

converts the integer value 128 to a string, then joins it together with the first string, to form the new string "The answer is 128".

Substrings

A substring is defined as a string that is part of a longer string. In Java, you can extract a substring from a larger string with the substring method of the String class. For example,

String fruit = "watermelon";
String s = fruit.substring(0,5);

creates a string consisting of the characters "water". Java counts the characters in strings in a peculiar fashion: the first character in a string has position 0.

For example, the character 'w' has position 0 in the string "watermelon", and the character 'm' has position 5. The second parameter of substring is the first position that you do not want to copy. In our case, we want to copy the characters in positions 0,1,2,3 and 4 (from position 0 to position 4 inclusive). As substring counts it, this means from position 0 inclusive to position 5 exclusive. For a visual description refer to diagram below.


To extract the substring "melon" you would use the following code. Remember the second parameter is one past the character you want to end on.

String fruit = "watermelon";
String s = fruit.substring(5,10);

The String class provides another substring method that takes only one parameter or argument. This number represents the starting position of the substring. This method automatically extracts all characters from the starting position to the end of the string. For example, to extract the substring "melon" you would use the following code.

String fruit = "watermelon";
String s = fruit.substring(5);    
// only need 1 parameter

You can use the substring method to extract an individual character from a string. For example,

String fruit = "watermelon";
String s = fruit.substring(0,1);  
// extracts 1st character

would extract the first character 'w' from the string.

String Editing

The String class provides a useful method named length that will tell you how many characters are in a string. In Java terminology, you are retrieving the string's length. For example,

String fruit = "watermelon";
int n = fruit.length();       
// length method sets n = 10

Strings are Immutable

The String class provides no methods that let you change a character in an existing string. If you want to turn fruit into "Watermelon" with a capital 'W', you cannot directly change the first position of fruit into a 'W'. How are you going to modify the string? In Java it is quite easy: take the substring that you want to keep, and then concatenate it with the characters that you want to replace.

fruit = "W" + fruit.substring(1,10);   
// fruit.substring(1,10) = "atermelon"

This changes the current value of the fruit variable to "Watermelon". I could have gotten the same result by using the length method:

fruit = "W" + fruit.substring(1, fruit.length());

Since you cannot change the individual characters in a Java string, the documentation refers to the objects of the String class as being immutable. Immutable means they cannot be changed. Just as the number 3 is always 3, the string "watermelon" will always contain the character sequence 'w','a','t','e','r','m','e','l','o','n'. You cannot change these values. You can, as you just saw, however, change the contents of the string variable fruit and make it refer to a different string, just as you can make a numeric variable currently holding the value 3 hold the value 4.

Searching a String

The String class provides a method named indexOf that will allow you to search for an individual character or substring within a string. If the search is successful the method returns the index(position) of the character or substring within the string.

Example 1

String str = "friend";
String index = str.indexOf("e");
System.out.println("Index = " + index);

The code above produces the following output:

Index = 3

The method returns a value of 3 because the string "e" was located at index 3 of the string "friend".

Example 2

String str = "friend";
int index = str.indexOf("x");
System.out.println("Index = " + index);

The code above produces the following output:

Index = -1

If the search key is not found within the string the method returns a value of -1.

Example 3

String str = "airplane";
int index = str.indexOf("air");
System.out.println("Index = " + index);

The code above produces the following output:

Index = 0

If the search key is a substring the method returns the index where the substring begins within the string. It returns -1 if the substring is not contained within the string.

Example 4

String str = "airplane";
int index = str.indexOf("a");
System.out.println("Index = " + index);

The code above produces the following output:

Index = 0

Notice that the string "airplane" contains two a's. The indexOf method will only return the first occurrence of a character or substring within a string.

Example 5

String str = "airplane";
int index = str.indexOf("a", 1);
System.out.println("Index = " + index);

The code above produces the following output:

Index = 5

The indexOf method allows you to include another parameter which tells the method where to begin the search. In this example the 1 after the search key tells the indexOf method to start its search for the "a" starting at index 1.
