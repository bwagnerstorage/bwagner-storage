Mixup

Directions

Write a Java program that allows a user to enter a string from the keyboard. Then use the String class's length and substring methods to rearrange the string so that the first half of the string is displayed second and the second half of the string is displayed first. You will also need to use the concatenation operator in your implementation.

Mixed up string = utercomp
