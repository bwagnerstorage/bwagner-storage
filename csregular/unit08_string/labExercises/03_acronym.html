Acronym

Directions

An acronym is an abbreviation of several words in such a way that the abbreviation itself forms a pronounceable word. Here are some examples of acronyms:

  • laser - light amplification by stimulated emission of radiation
  • scuba - self-contained underwater breathing apparatus
  • radar - radio detecting and ranging
  • RAM - random access memory

Write a program that allows a user to enter three words from keyboard then uses the first letter of each word to form an acronym.

Hints

  • use the Scanner class's next method to read the three strings from keyboard
  • use the String class's substring method to extract the first letter from each word
