Compare

Directions

Write a program that will allow a user to type in two integers from the keyboard then compare the integers to determine it the first integer is less than, greater than, or equal to the second integer. The output should look like the sample output below.

The use of methods is optional.

Source File

Compare.java

Sample Run 1

Enter Number 1 --> 3
Enter Number 2 --> 4

3 is less than 4

Sample Run 2

Enter Number 1 --> 10
Enter Number 2 --> 2

10 is greater than 2

Sample Run 3

Enter Number 1 --> 20
Enter Number 2 --> 20

20 is equal to 20
