LetterGrade

Directions

Write a program that reads a numerical grade from the user then prints a corresponding letter grade according to the scale below.

Numerical Grade Letter Grade
97 - 100 A+
94 - 96 A
90 - 93 A-
87 - 89 B+
84 - 86 B
80 - 83 B-
77 - 79 C+
74 - 76 C
70 - 73 C-
below 70 F

The use of methods is optional.

LetterGrade.java

Enter a Numerical Grade--> 87
Letter Grade = B+

Sample Run 2

Enter a Numerical Grade--> 71
Letter Grade = C-

Sample Run 3

Enter a Numerical Grade--> 50
Letter Grade = F
