SportsEvent

Directions

In many sports events, judges rate contestants with an average score being determined by discarding the lowest and highest scores and averaging the remaining scores.

Write a program which allows eight scores to be entered, discards the lowest and highest scores, then computes an average score for the contestant. Your program should display the eight entered scores, the average score, and the two discarded scores.

Java's Double class provides a useful constant that can be used when implementing the lowest method. The constant MAX_VALUE contains the largest value that can be represented in a double variable.
