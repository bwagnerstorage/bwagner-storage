WordList

Description

Write a program that reads 10 strings from the keyboard and stores them in an array of strings, then displays the list of strings vertically on the screen.

Directions

  1. Create a new Java file named WordList and add the usual starter code.
    import java.util.*;

public class WordList
{
  public static void main(String[] args)
  {
    Scanner keyboard = new Scanner(System.in);
    
  }
}
  2. In the main method, create an array named words that can store 10 strings.

  3. Assuming that you created the words array, the following code can be used to input 10 strings from the keyboard and store them in the array.
    words[0] = keyboard.nextLine();
words[1] = keyboard.nextLine();
words[2] = keyboard.nextLine();
words[3] = keyboard.nextLine();
words[4] = keyboard.nextLine();
words[5] = keyboard.nextLine();
words[6] = keyboard.nextLine();
words[7] = keyboard.nextLine();
words[8] = keyboard.nextLine();
words[9] = keyboard.nextLine();

    Although this code works, we can do alot better! In the first exercise you learned that you could use a for loop to display the contents of an array. In this exercise, lets use a for loop to input the 10 strings and store them in the array. Before we write the for loop, look again at the code above. Notice that the 10 lines are exactly the same accept for the number in the square brackets []. This number, called an index, starts with a value of 0 and increases by one until it reaches 9. This behavior can be duplicated by a for loop.
    for(int i = 0; i < 10; i++)
{
   words[
    i
    ] = keyboard.nextLine();
}

  4. Complete the program by writing a for loop to display the 10 strings.
