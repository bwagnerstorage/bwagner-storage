Directions

Complete the program below that allows a user to enter an unknown number of words and stores them in an array. Input should continue until the user types the sentinel value "stop". When the input process is complete, the program should display each word in its original form, in all upper case letters, and in all lower case letters.

Since you do not know how many words the user will enter the program needs to make the array's physical size large enough to accommodate a reasonable amount of words, therefore the program defines an array with a physical size of 50.

private String[] words = new String[50];   // array's physical size

The program also needs to include a variable that keeps track of the array's logical size, that is, the actual number of words that have been entered into the array.

private int size = 0;   // array's logical size

Method readWords

Complete the method readWords. This method should allow a user to enter words into the array until the sentinel value "stop" is encountered.

To enter multiple words into the array you will need to utilize a while loop. Inorder for the while loop to work properly you will need to define a local String variable within the method to store the user's input. You can then copy this variable's value into the array with code like the following:

str = keyboard.nextLine();
words[size] = str;         // size = logical size of the array

Make sure you increment the logical size of the array to reflect the addition of a new word to the list.

Method printWords

Complete the method printWords. This method should display each word in the array in its original form, in all upper case letters, and in all lower case letters.

The String class provides two methods for converting a string to all uppercase and lowercase letters. Look at the following example

String str = "Computer";
System.out.println(str.toUpperCase());
System.out.println(str.toLowerCase());

This code produces the following output

COMPUTER
computer

In Java strings are immutable. This means once a string has been defined it cannot be changed. Therefore the toUpperCase method returns a copy of the original string with all the letters converted to uppercase. The toLowerCase method returns a copy of the original string with all the letters converted to lowercase.
