VerticalString

Directions

Write a Java program that reads a string from the user and prints it vertically one character per line. Use a for loop in your implementation.

You will need to use the following methods:

Scanner class
 - nextLine

String class
 - length
 - substring

Review

To read a string value from the keyboard using the Scanner class you can use either the next() method or the nextLine() method. The difference between the two is that the nextLine() method will read until it encounters an end of line marker, which means it will read strings that contain spaces, where as the next() method will stop reading as soon as it encounters a space or end of line marker.

String str = keyboard.nextLine();

Recall that when your program stores a string in memory it assigns a number called an index to each character of the string; starting with the index 0. In previous assignments you used a String class method named substring to extract a character or characters from a string which we called substrings. Below is a table that demonstrates this process.

Suppose we create a String variable named str and assign it the string "horse".

String str = "horse";

Notice the call to the substring method. To extract each letter of the string, one at a time, involves using a distinct pattern: (0,1), (1,2), (2,3), (3,4), (4,5). The key to solving the problem of displaying the letters vertically one at a time is to use a for loop to emulate this pattern along with it's loop control variable to extract the individual letters from the string.

for Loop Condition and String class's length Method

The for loop below iterates or loops 11 times.

for(int i = 0; i <= 10; i++)

The condition statement for this loop will cause the loop to stop once the variable i is greater than 10. As it relates to the vertical string problem, what value do you use in place of the 10 to make the loop stop once the loop has visited every letter in a string? Thankfully, the String class provides a method named length for this purpose. The length method returns the number of characters in a string. For example, the string "horse" has 5 characters so the method would return a value of 5.

String str = "horse";
for(int i = 0; i < str.length(); i++)

Notice that because the for loop uses the less than symbol "<" the loop stops one short of the length of the string. Don't forget that the characters of a string are number starting with 0. The length of a string is the index value of the last character + 1.
