Compass

Directions

In this assignment, we are going to write a class that simulates the operation of a compass. Your compass will change directions by turning to the left or right one degree at a time or a specified number of degrees. A positive number will turn the compass needle to the right and a negative number will turn it to the left.

Using class decomposition we are going to decompose this problem into two classes. The first class will be named Compass. Its job will be to store the compass's current heading (direction). The second class will be named CompassManager. Its job will be to manage the program by creating a Compass object that it can send messages to asking it to update the compass heading. This class will contain the main method for the program.

Implement Compass Methods

  1. Implement the method turnRight. This method updates the instance variable heading by turning one degree to the right. After completing the turn, if heading is equal to 360 its value is changed to 0.

  2. Implement the method turnLeft. This method updates heading by turning one degree to the left. After completing the turn, if heading is equal to -1 its value is changed to 359.

  3. Implement the method turnDegrees. This method updates heading by turning the number of degrees specified by the value of the parameter deg. If deg is greater than or equal to 0 the method turns to the right deg degrees, otherwise it turns to the left deg degrees.
    • Use methods turnRight and turnLeft in your implementation.
    • Use for loops in your implementation to control how many times turnRight and turnLeft are executed.
    • Use the Math class's abs (absolute value) method to convert a negative deg value into a positive value.
      Example: Math.abs(deg) - the value returned by abs is always positive.

      Here is the pseudocode for the method turnDegrees: 
      if deg >= 0 then
   loop I from 0 to deg
      turnRight()
   end loop
else
   loop I from 0 to Math.abs(deg)
      turnLeft()
   end loop
end if

      Note: You will need to translate pseudocode into Java.


  4. Implement the method getHeading. This method returns the value of heading.

  5. Implement the method getDirection. This method translates heading into one of the eight directional values: North, NorthEast, East, SouthEast, South, SouthWest, West, and NorthWest.

    Use the following table to determine the directional value according to the value of heading.
    Degrees Direction
    (0 - 22) or (337 to 360) inclusive North
    (22 - 67) exclusive NorthEast
    (67 - 112) inclusive East
    (112 - 157) exclusive SouthEast
    (157 - 202) inclusive South
    (202 - 247) exclusive SouthWest
    (247 - 293) inclusive West
    (293 - 337) exclusive NorthWest

    6. Note: (inclusive means <= and >= ; exclusive means < and > )


    Programming Tip: Depending upon how you implement the method you may get the following compiling error: "missing return statement". You can fix this error by making the NorthWest direction the default for the method by removing its if statement. For example:

CompassManager class

The job of the CompassManager class is to manage the program by interacting with a Compass object. This class has been completed for you. You do not need to make any modifications to this class. However, make sure you examine the code to see how the CompassManager interacts with the Compass object.

Create a file named CompassManager.java and add the following code.

   import java.util.*;

public class CompassManager
{
    public static void main(String [] args)
    {
        
Compass compass = new Compass();

		
        Scanner keyboard = new Scanner(System.in);
        
        System.out.println("**********************");
        System.out.println("*    Compass Test     *");
        System.out.println("**********************");
        System.out.println();

        int sel = 0;
        
        do
        {
          System.out.println("================================");
          System.out.println(" Current Heading   = " + 
compass.getHeading()
);
          System.out.println(" Current Direction = " + 
compass.getDirection()
);
          System.out.println("================================");
          System.out.println();
          System.out.println("1. Turn Right One Degree");
          System.out.println("2. Turn Left One Degree");
          System.out.println("3. Turn (degrees)");
          System.out.println("4. Quit" );
          System.out.println();
          System.out.print("Make selection -->");
          sel = keyboard.nextInt();
          
          if(sel == 1)
          {
             
compass.turnRight();

          } 
          if(sel == 2)
          {
             
compass.turnLeft();

          }
          if(sel == 3)
          {
             System.out.println();
             System.out.print("Enter degrees (- turns left, + turns right) -->");
             int degrees = keyboard.nextInt();
             
compass.turnDegrees(degrees);

          }
        }
        while(sel < 4);
    }
}

Compile and run the program.
